Welcome to Sensi, your friendly super-simple Single Page App framework

This framework aims to simplify SPA development by using a jQuery as a "backend base" for DOM referencing and framework work-horsing ( such as html view loads ) alongside Vue's "frontend base" for templates, data injection and events ( and whatever else you want to do with Vue ). Furthermore it is built on top of Cordova for app deployment, however you can run it directly in Safari or Firefox (quickly, if you like, without a localhost) for ease and speed of development. To deploy and run in Cordova for your device, please follow the guide here: https://cordova.apache.org

Sensi is inspired by the classic Sensible Soccer video game, because we play it all the time for its purity and simplicity combined with just very, very good gameplay! Pick up and Play at its best. Hopefully this framework provides a similar kind of experience.

Sensi creates & appends views to the DOM and keeps them in memory as you request subviews, as you go back, these subviews are then destroyed. This is all done with a standard sliding page transition - to see how this works, try viewing the page source and clicking NEXT below.
